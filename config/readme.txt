Работа с конфигом.
vk_data_default.json, config_default.json И yandex_default.json - файлы который надо изменить. После изменения ОБЯЗАТЕЛЬНО надо изменить название на "vkdata.json".
group_token - это строка в которую идёт ключ доступа группы. Подробнее здесь: https://vk.com/dev/bots и https://vk.com/dev/bots_docs
owner_id - это номер ID основателя беседы или того, кто отвечает за работу бота.
responses - ответы на какие-то определённые строки. Ключ - это сообщение пользователя, а значение - как на него ответит бот.
com_symbol - символ для ввода команды. 
commands - список всех команд.
func - функция, которую вызовет определённое сообщение в коде. НЕ РЕДАКТИРУЙТЕ ЭТО ЕСЛИ НЕ РЕДАКТИРУЕТЕ КОД.
help - описание функции.
public - определяет, могут ли обычные пользоваетили использовать эту функцию.

greet - как боты встречают пользователя.
oauth_token - токен для бота Алисы
